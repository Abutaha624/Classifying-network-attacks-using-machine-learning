# Classifying-network-attacks-using-machine-learning
•	مشروع أكاديمي متكامل لتحليل حركة الشبكة

o	إحصائيات سريعة عن البيانات:
- عدد السجلات (Rows): 1,042,557 سجل.
- عدد الخصائص (Features): 79 عموداً تقنياً.
- الفئات: 4 فئات رئيسية (طبيعي + 3 أنواع هجمات).

•	هدف المشروع:
بناء نموذج ذكاء اصطناعي قادر على فحص بيانات حركة الشبكة وتصنيفها تلقائياً إلى حركة طبيعية أو نوع محدد من الهجمات السيبرانية. يساعد هذا النظام في بناء أنظمة كشف التسلل (IDS).

o	الفئات المستهدفة للتصنيف:
- BENIGN: حركة مرور طبيعية وسليمة.
- DoS (Denial of Service): هجمات حجب الخدمة.
- DDoS (Distributed DoS):هجمات حجب الخدمة الموزعة.
- PortScan: محاولات فحص المنافذ لاكتشاف الثغرات.
